# Shelby Shine Cleaning Service

Professional cleaning and property management across Kentucky. Standard, deep, specialized cleaning, roof washing, and solar panel care.

## Local Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Testing

Run tests with Playwright:
```bash
npx playwright test
```
